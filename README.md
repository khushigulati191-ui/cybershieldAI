# CyberShield

## Overview
CyberShield is a website and app security and privacy analyzer that evaluates websites using multiple security parameters and generates an overall trust score.

## Features

- URL Analysis
- Domain Age
- WHOIS Information
- SSL Certificate Validation
- HTTPS Check`
- DNS Checks
- Security Headers
- Cookie Analysis
- Website Reputation
- Security Score
- Privacy Score

## Technologies Used
